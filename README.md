# HR Analytics Dashboard

## Project Overview

This Power BI dashboard analyzes employee attrition, workforce demographics, job satisfaction, and HR performance metrics.

The dashboard provides insights into:

- Employee attrition trends
- Department-wise attrition analysis
- Job satisfaction analysis
- Gender distribution across age groups
- Education field attrition
- Workforce demographics
- Employee age group analysis
- Active employee tracking

---

## Key Insights

- R & D Department has the highest attrition
- Most attrition occurs in the 25–34 age group
- Life Sciences field has the highest attrition count
- Sales Executive role shows the highest dissatisfaction
- Male employees are higher across all age groups

---

## Dashboard Features

- Interactive slicers for education levels
- Dynamic KPI cards
- Donut charts for gender analysis
- Department-wise attrition visuals
- Job satisfaction heatmap
- Age group employee distribution
- Executive summary insights section
